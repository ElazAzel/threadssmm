# Обновления

## 2026-06-18

- Добавлена миграция стабилизации ядра: атомарный onboarding, резервирование AI-кредитов, persistent rate limit, транзакционные согласования и сохраняемые workspace-настройки.
- Подключена строгая типизация Supabase для frontend и serverless API; устранено небезопасное получение workspace через вложенное приведение типов.
- Усилены RLS-политики: `viewer` переведён в read-only, запись разрешена только `owner/admin/editor`, Storage следует тем же ролям.
- AI, RSS, Threads OAuth и publish endpoints получили rate limit и единый клиентский разбор сетевых/API-ошибок.
- Рабочими стали глобальный поиск, уведомления, «Что нового», политики настроек, реальный audit log с CSV, история AI Studio и варианты визуального промпта.
- Доработаны аккаунты (назначение бренда, удаление с подтверждением), медиатека (выбор бренда), мониторинг (удаление источника/скрытие материала) и approval editor.
- Календарь переведён на реальные даты, фильтры и режимы; исправлен desktop overflow на 1280 px.
- Расширены Playwright-тесты до 10 сценариев. `npm run verify` проходит полностью, `npm audit --omit=dev` сообщает `0 vulnerabilities`.

## 2026-06-17

- Добавлен `docs/product-roadmap.md` с большим планом развития платформы, фазами реализации и чекбоксами прогресса.
- Реализована диагностика окружения для Vercel/Supabase/Gemini/Threads без раскрытия секретов: добавлен `/api/health`, обновлен `/setup`, расширены Playwright smoke-тесты.

## 2026-06-15

- Собрано единое React/Vite-приложение на основе desktop и mobile Stitch-макетов.
- Добавлены лендинг, onboarding и 10 разделов рабочего пространства.
- Реализованы локальные сценарии генерации, compliance-проверки, согласования и планирования.
- Добавлена адаптивная мобильная навигация и отдельные mobile-композиции ключевых экранов.
- Добавлены Vercel SPA rewrite, ESLint, TypeScript и команда `npm run verify`.
- Build toolchain обновлён до Vite 8; `npm audit` не обнаруживает уязвимостей.
- Добавлены Playwright smoke-тесты прямых SPA-маршрутов, полного approval-сценария и мобильной адаптивности.
- Добавлен GitHub Actions workflow для автоматической проверки перед публикацией через Vercel.
- Репозиторий подключён к Vercel, production опубликован на `https://threadssmm.vercel.app`.
- Production smoke-проверка подтвердила прямые SPA-маршруты, approval flow и мобильную адаптивность без runtime-ошибок.
